1. 앞으로 git 프로젝트로 만들고 싶은 것
 - 개발일지 작성하기 : 오늘 배운것들에 대한 내용 정리해보기
 - 에러노트 작성하기 : 오늘 있었던 개발 내용중에 에러내용 적어보고 이유 예측해보고 실제로 풀어낸 방법 비교 정리하기.

2. 1주차 주요 개념 키워드 적어보기
 - git 이 뭘까?
 - github 은 뭘까?
 - commit 이란?
 - 나의 로컬 프로젝트 git 으로 생성하기
 - 스테이지에 올리고 commit 하기
 - 여러개 올리고 commit 하기
 - 원격 repo 란?
 - 로컬 repo 란?
 - 원격 repo 생성하기
 - 로컬 repo 와 원격 repo 연결하기
 - 로컬 repo 의 내용 원격 repo 에 push
 - 원격 repo 의 내용 로컬 repo 에 pull(당겨오기)
 - 텅빈 로컬 repo 에 원격 repo 전체 내용 clone 해오기
